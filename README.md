# AWS S3 Operations

This repository contains a package for various i/o functions on s3 buckets.

### To install

`pip install aws_s3_ops`

### To uninstall

`pip uninstall aws_s3_ops`

### Usage

The list of available functions are:

- `save_pickle`
- `load_pickle`
- `save_csv`
- `save_json`
- `download_file`
- `upload_file`
- `key_exists`
- `delete_data`
- `get_prefix_object`
- `get_file_buffer`
